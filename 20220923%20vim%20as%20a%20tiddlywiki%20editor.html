20220923 vim as a tiddlywiki editor

9th February 2023 at 11:35am

During the last week, I was obsessing over converting vim (more specifically gVim) to a tiddlywiki files specific editor.

This has been an idea I had for a long time, not having to fire up my browser to quick find some info from my wiki.

I have to report that everything went smooth, thanks to the vim-tiddlywiki plugin which I had to tune up a bit to run on windows and linux. I also created a set of mappings and wrote a couple of functions that didn't run well with the plugin to make the process easier.

Another useful plugin I used is ctrlp which allows me to find tiddlers easily.
