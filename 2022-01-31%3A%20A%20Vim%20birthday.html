2022-01-31: A Vim birthday

2nd April 2023 at 11:07am

Today it's my birthday, and what would be a better present than my new automated VimEditor tiddler creation macros?

It was not easy doing this, I've been trying for 3 days and finally I did it adding the custom commands inside my _vimrc in Windows. This way, I will be able to edit my wiki without having to open a browser, and using my good old vim editor.

Of course, for the whole thing to work I need the TiddlyWiki vim plugin for syntax highlighting (it's other functions dont quite work for Windows and that's why I had to create those macros), adding the Tiddlywiki path in _vimrc and using the :find command to search for tiddlers. I think it works nice so far, but we shall see.

I'm using F4 to create or open the today's journal, and F5 to insert the Tiddlywiki headers. Later in the week, I will review my notes and update my tiddlywiki normally from the browser. This is just for quick notes.
