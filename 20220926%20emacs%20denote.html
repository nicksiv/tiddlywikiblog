20220926 emacs denote

9th February 2023 at 11:39am

I tried this package from protesilaos, for Zettelkasten type of notes. It is simple and nice, with a specific file naming scheme, great front matter generators for md, org and even txt files. It supports links in all formats and even has a function to show backlinks. It doesn't contain much and you need to add custom mappings and stuff by reading the documentation.

https://protesilaos.com/emacs/denote : Denote documentation

I got ideas from this about using dired to view and search for files. I've been using deft for this kind of functionality but I guess I can do without it.

The package feels a bit incomplete targetting experienced users but it's simple and well made.
